# Boardgame "Comersant"

## Server and Client

***

To run test instance use "Gulp"

```
git clone https://github.com/senyaak/expressComersant.git

npm install

typings install (if not installed "npm install typings -g")

gulp watch (if not installed "npm install gulp -g")

```
